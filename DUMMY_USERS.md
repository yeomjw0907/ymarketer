# 더미 회원 계정 정보

테스트용 더미 회원 계정입니다. 아래 정보로 회원가입하거나 Supabase Dashboard에서 생성해주세요.

## 생성 방법

### 방법 1: 회원가입 페이지 사용 (권장, 가장 간단)

1. http://localhost:3000/signup 접속
2. 아래 정보로 회원가입
3. 자동으로 `auth.users` 및 `profiles` 테이블에 저장됨

### 방법 2: Supabase Dashboard 사용

1. Supabase Dashboard → **Authentication** → **Users**
2. **"Add user"** 버튼 클릭
3. 이메일/비밀번호 입력 후 생성
4. 생성된 사용자의 프로필 정보는 로그인 후 마이페이지에서 수정하거나, SQL로 직접 업데이트
