## 🎨 1. 디자인 철학

### 핵심 컨셉
**"Less is More" - 미니멀 럭셔리**

- **무신사**: 깔끔한 화이트 베이스 + 블랙 텍스트, 그리드 중심
- **크림**: 프리미엄 스니커즈 플랫폼, 하이엔드 느낌, 큰 이미지
- **ymarketer**: 일본 직구 플랫폼의 신뢰성 + 세련된 감각

## 🎯 2. 색상 시스템 (Color System)

### Primary Colors (주색상)
```
Black (블랙)
- #000000 (Pure Black) - 헤더, 주요 텍스트, CTA 버튼
- #1A1A1A (Soft Black) - 서브 헤딩
- #2D2D2D (Charcoal) - 카드 배경 (다크모드 대비용)

White (화이트)
- #FFFFFF (Pure White) - 메인 배경, 카드
```

```
Gray Scale (그레이 스케일)
- #666666 - 서브 텍스트
- #999999 - Placeholder, 비활성

- #E5E5E5 - 구분선
```

## 📐 3. 타이포그래피 (Typography)

### 폰트 전략
```css
/* 메인 폰트 - 유지 */
font-family: "Pretendard", -apple-system, sans-serif;

/* 폰트 웨이트 체계 */
- Regular (400): 본문 텍스트
- Medium (500): 서브 타이틀
- SemiBold (600): 섹션 제목
- Bold (700): 브랜드명, CTA
- ExtraBold (800): 히어로 타이틀
```

### 폰트 크기 체계 (Desktop / Mobile)
```
Hero Title: 48px / 32px (font-bold)
Section Title: 28px / 20px (font-bold)
Product Name: 14px / 13px (font-medium)
Body Text: 14px / 13px (font-regular)
Small Text: 12px / 11px (font-regular)
Micro Text: 10px / 10px (font-regular)
```

### 행간 (Line Height)
```
타이틀: 1.2
본문: 1.5
상품명: 1.4
```

## 🖼️ 4. 레이아웃 & 스페이싱

### 그리드 시스템 (무신사 스타일)
```
Desktop (1440px 기준)
- Container: 1200px (max-width)
- Gutter: 24px
- 상품 그리드: 4~5개 (240px per card)

Tablet (768px ~ 1024px)
- Container: 100%
- Gutter: 16px
- 상품 그리드: 3~4개

Mobile (~ 767px)
- Container: 100%
- Gutter: 16px
- 상품 그리드: 2~3개
```

### 스페이싱 (8px 기준)
```
Micro: 4px (0.25rem)
Small: 8px (0.5rem)
Medium: 16px (1rem)
Large: 24px (1.5rem)
XLarge: 32px (2rem)
XXLarge: 48px (3rem)
Section: 80px (5rem) - Desktop
Section: 48px (3rem) - Mobile
```

### 보더 반경 (Border Radius)
```
None: 0px - 크림 스타일 (각진 디자인)
Small: 2px - 버튼, 배지
Medium: 4px - 카드, 입력 필드
Large: 8px - 모달
```

**제안: 무신사는 둥근 모서리, 크림은 각진 디자인**
→ **ymarketer는 크림 스타일의 각진 디자인 채택 (0~4px)**

```
검색바: 배경 #F5F5F5, 보더 없음

```

**레이아웃:**
```
[로고]               [검색바 (중앙)]               [알림 장바구니 마이]
```

### 5.2 상품 카드 (Product Card)
```
배경: #FFFFFF
보더: 없음 (hover 시 1px solid #E5E5E5)
그림자: 없음 (hover 시 0 4px 12px rgba(0,0,0,0.08))

간격: 8px (모바일), 16px (데스크톱)
```

**정보 레이아웃:**
```
[이미지 1:1]
  - 할인율 배지 (좌상단): #FF0000, 10px bold

[브랜드명] - #999999, 10px, uppercase
[상품명] - #000000, 13px, medium, 2줄 말줄임
[가격] - #000000, 14px, bold
[원가] - #999999, 11px, line-through
```

## 📱 6. 페이지별 리디자인 계획

```

```

#### 카테고리 섹션
```
현재: 아이콘 + 색상 배경
변경:
- 아이콘 제거 또는 블랙 아이콘만
- 배경: 화이트
- 보더: 1px solid #E5E5E5
- 텍스트: #000000, Bold
- Hover: 배경 #FAFAFA
```

#### 상품 그리드
```
현재: 3열 (모바일), 다양한 색상
변경:
- 2~3열 (모바일), 깔끔한 간격
- 화이트 배경, 블랙 텍스트

- 최소한의 정보만 표시
```

### 6.2 상품 상세 페이지 (Product Detail)

#### 레이아웃
```
현재: 2열 (이미지 | 정보)
변경: 크림 스타일
- 좌측: 큰 이미지 (1:1 비율, 여백 없음)
- 우측: 정보 (스티키)
  - 브랜드명 (작게, 그레이)
  - 상품명 (크게, 볼드)
  - 가격 (매우 크게)
  - 구매 버튼 (풀 너비, 블랙)
```

#### 가격 표시
```
현재: 여러 가격 정보
변경:
- 일본 구매가: 크게, 블랙, 볼드
- 한국 가격: 작게, 그레이, 취소선
- 절약 금액: 작게, 레드
```

### 6.3 카테고리 페이지

#### 필터 & 정렬
```
위치: 상단 스티키
배경: #FFFFFF
보더: 하단 1px solid #E5E5E5

필터 버튼: 블랙 보더, 화이트 배경
정렬 드롭다운: 텍스트만 (버튼 없음)
```

### 6.4 마이페이지

#### 메뉴
```
현재: 카드형, 아이콘 + 색상
변경:
- 리스트형 (크림 스타일)
- 왼쪽: 텍스트만 (블랙)
- 오른쪽: 화살표 (→)
- 보더: 하단 1px solid #E5E5E5
```

## 🎭 7. 무신사 vs 크림 비교 분석

### 무신사 스타일
**특징:**
- 밝은 화이트 배경
- 둥근 모서리 (8px~12px)
- 다양한 컬러 포인트
- 친근한 느낌
- 정보량 많음

**장점:**
- 접근성 좋음
- 다양한 카테고리 표현 가능
- 부드러운 느낌

**단점:**
- 프리미엄 느낌 부족
- 차별화 어려움

---

### 크림 스타일 ⭐ 추천
**특징:**
- 화이트/블랙 대비
- 각진 디자인 (0~2px radius)
- 최소한의 색상 사용
- 큰 이미지, 큰 타이포그래피
- 여백 중시
- 정보량 최소화

**장점:**
- 프리미엄 브랜드 이미지
- 세련되고 모던함
- 상품 이미지 집중
- 차별화된 아이덴티티

**단점:**
- 차가운 느낌 (친근함 부족)
- 정보 전달력 약할 수 있음

## 🎯 8. ymarketer 전략 제안

### 추천 방향: **크림 스타일 베이스 + 무신사 실용성**

**이유:**
1. **차별화**: 일본 직구 플랫폼의 프리미엄 이미지
2. **신뢰성**: 블랙 앤 화이트의 고급스러움
3. **상품 집중**: 가격 비교가 핵심이므로 숫자 강조
4. **국제적 감각**: 일본 브랜드와의 연결성

### 하이브리드 전략
```
크림에서 가져올 것:
- ✅ 블랙 앤 화이트 색상
- ✅ 각진 디자인 (0~4px radius)
- ✅ 큰 타이포그래피
- ✅ 최소한의 색상 (레드만)
- ✅ 여백 중시

무신사에서 가져올 것:
- ✅ 친근한 카피라이팅
- ✅ 명확한 정보 전달
- ✅ 카테고리 구분
- ✅ 실용적 필터/정렬
```
